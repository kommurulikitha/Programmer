Remaining methods on str and array

Replace in string:

The replace() method replaces a specified value with another value in a string.

Substring in string:

Substring is similar to slice.But it won't accept negative indexes.

Trim in string:

The trim() method removes whitespace from both sides of a string:

Substr in string:

substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.

Splicing an Array:

The splice() method can be used to add new items to an array and can also used to delete some items in an array.

Slicing an Array:

The slice() method slices out a piece of an array into a new array.It is similar to splice()
