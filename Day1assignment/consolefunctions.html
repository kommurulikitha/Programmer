There are many types of console functions.As many webs uses this for debugging the bugs.The Console object provides access to the browser's debugging console.They are: 1.console.assert(): Writes an error message to the console if the assertion is false. 2.console.clear(): It refreshes or clears the console. 3.console.count(): Log the number of times this line has been called with the given label. 4.console.countReset(): It resets or start from beginning the value of the counter with the given label. 5.console.debug(): It give the output message to the console with the log level "debug". 6.console.dir(): It displays an interactive listing of the properties of a specified JavaScript object. This listing lets you use disclosure triangles to examine the contents of child objects. 7.console.dirxml(): Displays an XML/HTML Element representation of the specified object if possible or the JavaScript Object view if it is not possible. 8.console.error(): It outputs an error message. You may use string substitution and additional arguments with this method. 9.console.exception(): It is as same as error because exception will be handled when error ocuurs.An alias for error(). 10.console.group(): It creates a new inline group, indenting all following output by another level. To move back out a level, call groupEnd(). 11.console.groupCollapsed() Creates a new inline group, indenting all following output by another level. However, unlike group() this starts with the inline group collapsed requiring the use of a disclosure button to expand it. To move back out a level, call groupEnd(). 12.console.groupEnd() Exits the current inline group. 13.console.info() Informative logging of information. You may use string substitution and additional arguments with this method. 14.console.log() For general output of logging information. You may use string substitution and additional arguments with this method. 15.console.profile() Starts the browser's built-in profiler (for example, the Firefox performance tool). You can specify an optional name for the profile. 16.console.profileEnd() Stops the profiler. You can see the resulting profile in the browser's performance tool (for example, the Firefox performance tool). 17.console.table(): Displays tabular data as a table. 18.console.time(): Starts a timer with a name specified as an input parameter. Up to 10,000 simultaneous timers can run on a given page. 19.console.timeEnd(): Stops the specified timer and logs the elapsed time in seconds since it started. 20.console.timeLog(): Logs the value of the specified timer to the console. 21.console.timeStamp(): Adds a marker to the browser's Timeline or Waterfall tool. 22.console.trace(): Outputs a stack trace. 23.console.warn(): Outputs a warning message. You may use string substitution and additional arguments with this method.
